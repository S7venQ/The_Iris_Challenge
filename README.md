# 🌸 极具挑战性的鸢尾花数据分类与可视化 🌸

## 📊 项目简介

> **🎓 欢迎来到鸢尾花分类与可视化的多维探索之旅！** 🌍  
> 本实验项目深入挖掘经典Iris数据集，通过一系列精心设计的可视化技术，将机器学习模型的“黑箱”决策过程转化为直观、美观的视觉表达。  
> 从二维平面到三维空间，从单一模型到多算法对比，我们带你穿越特征空间，揭开分类器决策边界的神秘面纱！ 🔍✨  

## ✨ 项目特色

### 🎨 多层次可视化系统
- **📈 二维决策边界**：四类算法对比，十六个子图矩阵，直观展示不同分类器的“思考方式”
- **🌌 三维决策平面**：在三维特征空间中构建最优分割平面，可视化两类分离过程
- **🎭 三维概率映射**：将概率分布转化为点云艺术，感受模型置信度的空间变化
- **🔄 综合比较系统**：八大算法对比，多维性能评估，全面解析模型特性

### 🔧 模块化架构设计
- **🎯 独立运行模块**：七个功能模块可单独运行，满足不同探索需求
- **🎛️ 统一主控界面**：一键启动，线性流程，用户友好交互体验
- **💾 自动化保存**：所有可视化结果自动保存为高清图片，便于报告撰写和展示

### 📊 全面性能分析
- **🎯 准确率对比**：训练集与测试集双维度评估
- **📉 过拟合分析**：雷达图直观显示模型泛化能力
- **🔥 混淆矩阵热图**：可视化分类错误模式
- **⚖️ 特征重要性**：揭示不同特征对决策的贡献度

### 🌈 美学与实用并重
- **🎨 精心配色方案**：金色、绿色、蓝色分别代表Setosa、Versicolor、Virginica
- **📐 科学可视化布局**：子图排列有序，信息密度高而不乱
- **🔍 交互式探索**：Plotly图表支持缩放、旋转、悬停查看详情

## 🚀 快速开始

### 📋 环境要求
- **Python 3.7+** 🐍
- **核心依赖**：
  ```
  numpy
  matplotlib
  scikit-learn
  seaborn
  pandas
  plotly
  ```

### 📥 一键安装
```bash
# 安装所有必要依赖
pip install numpy matplotlib scikit-learn seaborn pandas plotly
```

### 🎮 启动项目
```bash
# 方式一：使用主控界面（推荐）
python main.py

# 方式二：单独运行特定模块
python mission1_reproduce.py      # 运行任务一
python n1_mission1_extend.py      # 运行扩展功能一
```

## 🎯 模块功能介绍

| 模块 | 功能描述 | 可视化输出 |
|------|----------|------------|
| **数据探索** | Iris数据集基本统计分析，特征分布可视化 | 箱线图、交互式散点图 |
| **任务一** | 四种分类器在二维特征空间的决策边界对比 | 4×4子图矩阵，十六个可视化面板 |
| **任务二** | 三维特征空间中的最优分割平面 | 3D决策平面与数据点分布 |
| **任务三** | 三维概率场可视化，Setosa类别高概率区域 | 3D概率点云+XY平面投影 |
| **扩展一** | 八大分类器全面比较，多维性能分析 | 决策边界图、准确率对比、混淆矩阵 |
| **扩展二** | 3D决策边界与概率映射的集成可视化 | 双面板3D可视化，边界与概率同屏展示 |

## 📊 分类器对比概览

| 分类器 | 核心思想 | 可视化特点 |
|--------|----------|------------|
| **样条特征+逻辑回归** | 非线性特征变换 + 线性分类 | 平滑的概率梯度变化 |
| **分箱特征+逻辑回归** | 离散化特征 + 线性分类 | 块状决策区域 |
| **梯度提升树** | 集成学习，多弱分类器组合 | 复杂但清晰的决策边界 |
| **RBF核SVM** | 核技巧，高维空间线性可分 | 辐射状概率分布 |
| **逻辑回归（扩展）** | 经典线性分类器 | 简洁的线性边界 |
| **K近邻** | 基于距离的懒惰学习 | 局部敏感的决策区域 |
| **随机森林** | 多决策树集成 | 细腻的边界细节 |
| **神经网络** | 多层感知器，非线性映射 | 最复杂的决策模式 |

## 🔧 技术亮点

### 🎨 可视化技术
- **📐 网格采样算法**：高效生成特征空间采样点
- **🌈 自定义色彩映射**：从白色到类别色的渐变，直观显示概率强度
- **🌌 3D渲染优化**：透明表面、点云抽样、视角控制，平衡美观与性能
- **📊 多子图同步**：坐标轴对齐，便于跨分类器比较

### 🤖 机器学习技术
- **🔧 特征工程**：样条变换、分箱离散化、标准化预处理
- **🎯 模型评估**：分层抽样、训练测试分割、过拟合量化分析
- **⚙️ 超参数优化**：合理的默认参数设置，保证模型良好表现

### 🏗️ 软件工程
- **🧩 模块化设计**：每个功能独立封装，高内聚低耦合
- **🎛️ 用户友好界面**：主控程序统一调度，避免命令行复杂操作
- **💾 自动化工作流**：数据加载→模型训练→可视化→保存，全自动流程
